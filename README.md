**"Wah"** is an appreciation term in Urdu language which is usually said is different contexts, but in poetry events a lot. In this repo, I have used simple template matching to find reference audio in any audio and the results are acceptable.

### Why acceptable?
this repo can break in very fine terms and in edge cases but it will work as bruteforce cheap method anyway. since I also speak urdu as my third language, i won't call the template a gold-standard. as pointed in the colab notebook, the code works and provided an answer of 23 over demo audio.
